Breakpoint
Simple helper for @media queries. Using scss list for breakpoints difinition.

Installation

npm install --save @sfem/breackpoint

Usage

Import module to your scss

@import '@sfem/breakpoint';

Call mixins:

@include bp(xs) {
  // code
}

@include bpq('min-width 20em') {
  // code
}

Breakpoints:

$breakpoints: (
  xxs: 320px,
  xs: 640px,
  s: 768px,
  m: 1024px,
  l: 1280px,
  xl: 1440px,
  xxl: 1600px
) !default;
